Admin panel setup

- After buying the webhosting, come to the Cpanel, than go to "MySQL® Databases"

- after that create database by choosing database name.

- after that add new user by choosing username and password.

- after that add user to database.

- tick all checklist.

- Now Upload digitalposter.sql file on your created database by going to myphpadmin.

- Now go to file manager of cpanel and upload ‘digitalposter.zip’ folder in your cpanel file manager in public.html/digitalposter. (as your location)

- Extract ‘digitalposter.zip’

- Now Go to application/config/config.php and Change configuration $config['base_url'] as per localhost or example.com & save file.

- Now Go to application/config/constants.php and Change configuration Constants as per localhost or example.com & save file.

- Now Go to application/config/database.php and Change configuration username, password, database & save file.

By going to "your project location" link for example "example.com/digitalposter" change example.com with your domain(depend on your url).

  • Now You can Login to Admin Panel with email and password.
  • Now You can chnage anything as per your requirement.
  • Note: If you’re experienced at cpanel and all, have configured & setup any other admin panel then you can do it. Otherwise if you receive any error, please mail us at arya.infotech3360@gmail.com

Replace Ad Id

Replace Ad Id
  • res Folder > values Folder > Strings.xml
  • BannerAdId = Google Banner Ad Id
  • FullAdId = Google Full Ad Id
  • FBBannerAdId = Facebook Banner Ad Id
  • FBFullAdId = Facebook Full Ad Id

Change Application Icon

Change AndroidManifest File > android:icon="@drawable/icon" Replace icon from drawable folder res Folder > drawable Folder > icon.png

Now Generate Signed Apk

Follow this Step When Create New Key Store:

Android Studio >

Build >

Generate Signed Bundle/APK >

Select APK >

Click on Next >

Create New Key store >

Fill All Details to Create New key store >

Enter Key store password >

select key alias >

key password >

Click on Next >

In Build Variants Select Release >

In Signature Versions Select V1(Jar Signature) >

Click on Finish.

Android Studio >

Generate Signed Bundle/APK >

Select APK >

Click on Next >

Go to the key store Path >
